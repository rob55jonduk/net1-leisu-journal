# net1-leisu-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能涉及不同主题、不同时期的内容，彼此之间没有固定的域名或网站关联。仓库的主要目的是集中存放这些页面，方便版本管理和访问。

## 项目简介

`net1-leisu-journal` 是一个静态的 HTML 页面集合仓库。每个页面通常是一个独立的文档、工具或演示页面，可以单独打开使用。仓库本身不提供任何网站或服务，仅作为页面文件的存储与发布载体。

## 目录结构

```
.
├── index.html          # 仓库首页（可选）
├── page-1.html         # 示例页面
├── page-2.html         # 示例页面
├── assets/             # 公用资源文件（CSS、JavaScript、图片等）
│   ├── style.css
│   └── images/
└── README.md           # 本文件
```

实际页面和资源文件可能根据归档内容有所不同。建议通过仓库文件列表查看具体页面。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖其他页面。
- 页面可能引用 `assets/` 目录下的公共资源，以保证样式和功能的一致性。
- 页面内容可能随时间更新或替换，历史版本可通过 Git 提交记录查看。
- 部分页面可能包含实验性或临时性内容，不保证长期可用。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改页面。
- 欢迎通过 Issue 提交问题或建议，但请理解仓库定位为个人归档项目，不一定接受所有外部请求。
- 不建议直接依赖本仓库的页面作为生产环境资源，因为结构和路径可能发生变化。
- 如需引用或分发，请遵守仓库使用的开源许可证（见 LICENSE 文件）。

## 使用方式

1. 直接访问 GitHub Pages 或 Raw 文件链接查看页面。
2. 克隆仓库到本地，用浏览器打开任意 HTML 文件。
3. 通过 GitHub 文件列表浏览所有页面。
